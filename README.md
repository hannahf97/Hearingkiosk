# Hearingkiosk
[캡스톤디자인] AI 음성인식을 활용한 키오스크 앱 
개발 인원 : 3명

## 작품 개요

Hearing Kiosk만의 특징

● 얼굴 인식 기능을 통해 시각장애인이 키오스크의 위치를 확인하고 주문을 시작할 수 있도록 음성으로 안내한다.
● 인공지능 음성인식 기능을 통해 버튼 조작이 어려운 디지털 소외계층이 키오스크와 대화를 나누며 쉽게 메뉴를 주문할 수 있게 한다.
